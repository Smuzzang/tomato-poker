# 토마토 포커 (Tomato Poker)

Tomato Arcade 등록용 웹 포커 게임. 순수 HTML/CSS/JS, 빌드 과정 없음, GitHub Pages 호스팅.

- 라이브: https://smuzzang.github.io/tomato-poker/
- 자매 프로젝트: [토마토 맞고](https://github.com/Smuzzang/tomato-matgo)

## 구조 (예정)
- `index.html` — 진입점
- `css/style.css` — 스타일
- `js/` — 게임 로직 (덱·핸드 평가·진행·AI·온라인)

## 현재 상태
스캐폴드 단계. 게임 방식(텍사스 홀덤 / 비디오 포커 등) 기획 확정 후 구현 예정.
